# ICOM Web Dashboard

React 기반 웹 대시보드 (Neon-Glass Protocol Design System)

## 특징

- 실시간 데이터 시각화
- 수요 예측 및 광고 최적화
- AI 에이전트 통합 UI
- Responsive design

## 기술 스택

- React
- Recharts (데이터 시각화)
- Lucide Icons
- Tailwind CSS (implied)

## 개발 모드 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
